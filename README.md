### 학습을 위한 요구사항 
- [] 스터디(study)를 생성한다. 
  - [] 주제(topic)을 가진다.
    - [] 주제는 변경이 가능하다.
  - [] 스터디는 진행 상황을 가진다.
    - [] 모집중(default)  
    - [] 진행중
    - [] 종료됨 
    - [] 진행 상황은 변경이 가능하다.
  - [] 스터디 목록을 확인할 수 있다.
